# CV Generator App

A dynamic Curriculum CV builder web application created as part of The Odin Project's React curriculum. This application allows users to input their personal details, educational background, and practical work experience through intuitive forms, dynamically rendering a clean, professional CV in real-time.

It was very challaneging to take this on as a first React project, however I got pretty good concpets of how state hook works. so win win?

## Live Demo

[View the Live Project](https://cvbuilds.netlify.app/)[cite: 1]

## Features

- Real-time Live Preview: See changes update instantly on the CV sheet as you type[cite: 1].
- Dynamic Educational and Practical Experience Forms: Easily add, edit, or delete multiple education credentials and work history items[cite: 1].
- Form Validation and Modes: Toggle cleanly between editing mode and a finalized Preview/Submit mode[cite: 1].
- Clean and Responsive Layout: Minimalist, professional styling designed to mimic a real resume document[cite: 1].

## Tech Stack

- Frontend: React.js, JavaScript (ES6+)[cite: 1]
- Build Tool: Vite[cite: 1]
- Styling: CSS3[cite: 1]

## Installation and Setup

1. Clone the repository:

   ```bash

   ```

````[cite: 1]

2. Navigate into the project directory:
   ```bash
   cd CVApp

```[cite: 1]

```[cite: 1]

4. Start the local development server:
   ```bash
````
